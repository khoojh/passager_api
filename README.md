# passager_api

API for passager

## Getting Started

### Prerequisites

#### Node

[Node](http://nodejs.org/) is really easy to install & now include [NPM](https://npmjs.org/).

### Install

    $ git clone https://github.com/khoojh/passager_api.git
    $ cd passager_api
    $ npm install

### Start & watch

    $ npm start

## Built With

* [ExpressJS](https://expressjs.com/)
